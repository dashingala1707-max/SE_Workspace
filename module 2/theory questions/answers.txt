1. Overview of C Programming
History, Evolution, Importance and Use of C

Introduction
C is one of the most popular and powerful programming languages used in computer science. It is a general-purpose programming language that is widely used for system programming and software development.

History and Evolution
The C programming language was developed by Dennis Ritchie in 1972 at Bell Laboratories. It was created to improve the B programming language, which itself was derived from the BCPL language.

C was mainly designed to develop the Unix Operating System, which made it very popular. Over time, C became the foundation for many other programming languages such as C++, Java, and C#.

10.String Handling Functions in C
Strings in C are arrays of characters ending with the null character \0. C provides several functions in the C Standard Library (string.h) to handle strings.
some of the keywords are 
strlen(): Used to find the length of a string.
strcmp(): Used to compare two strings.
strcat(): used to join (concatenate) two strings.
strcpy(): Used to copy one string into another.
